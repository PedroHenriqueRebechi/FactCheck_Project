Como o sistema foi desenvolvido?

Esta plataforma foi desenvolvida com foco em combater a desinformação de forma simples e acessível, utilizando tecnologias modernas e confiáveis.


Quando você insere uma dúvida, notícia ou frase que deseja verificar, o sistema faz uma requisição à Google Fact Check API, uma ferramenta oficial do Google que reúne verificações feitas por organizações especializadas no mundo todo.


A aplicação foi construída com as seguintes tecnologias:

  • HTML, CSS e JavaScript: para criar uma interface leve, responsiva e interativa.
  • Google Fact Check API: responsável por buscar os resultados das checagens de fatos.
  • Fetch API: utilizada para conectar o site à API e trazer os dados em tempo real.

A resposta da API retorna dados como:

  • A afirmação que foi verificada
  • Quem realizou a checagem (Fonte da informação)
  • A classificação do fato (verdadeiro, falso, enganoso etc.)
  • Link para a fonte completa
